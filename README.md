# cloud-workshop-adventure

This repository contains two main applications: `quiz-ui` and `quiz-api`.

## Applications

### quiz-ui
The `quiz-ui` is a frontend application built to provide an interactive user interface for the quiz platform. It allows users to:
- Take quizzes.
- View quiz results.
- Navigate through a user-friendly interface.

#### Technologies Used
- React 
- tailwind for styling
- typescript

#### Setup Instructions
1. Navigate to the `quiz-ui` directory.
2. Install dependencies using `pnpm install`.
3. Start the development server using `pnpm run dev`.

---

### quiz-api
The `quiz-api` is a backend application that powers the quiz platform by providing RESTful APIs for:
- Managing quizzes and questions.
- Storing and retrieving quiz results.
- Static User authentication.

#### Technologies Used
- Node.js with Express

#### Setup Instructions
1. Navigate to the `quiz-api` directory.
2. Install dependencies using `pnpm install`.
4. Start the server using `pnpm run dev`.

---

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/yogesh-kumarasamy/cloud-workshop-adventure.git
   ```
2. Follow the setup instructions for both `quiz-ui` and `quiz-api`.
